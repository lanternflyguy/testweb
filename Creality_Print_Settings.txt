POST-PROCESSING REQUIREMENTS:
=============================
1. Remove supports carefully
2. Sand threaded areas smooth
3. Drill/tap threads to final size:
   - M10 x 1.5 (pressure relief)
   - M12 x 1.75 (spring housing)
   - 1/4" NPT (pressure gauge)
   - 1/2" NPT (main connections)
4. Test fit all components
5. Acetone vapor smooth (optional)

PRINT ORDER RECOMMENDATION:
==========================
1. Pressure Gauge Housing (test print)
2. Flow Control Valve (test threads)
3. Trigger Valve Assembly (critical precision)
4. Injection Probe Handle (main component)
5. Backpack Mount Frame (overnight print)
6. Main Reservoir Tank (final component)

TROUBLESHOOTING:
===============
- Stringing: Reduce temperature by 5°C
- Poor overhangs: Add more support
- Warping: Increase bed temperature to 65°C
- Thread issues: Print 0.1mm oversized, then tap
- Layer adhesion: Increase nozzle temp by 5°C
